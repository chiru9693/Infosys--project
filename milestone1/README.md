# Parallel Text Handling Processor
# Milestone1 

## 📌 Project Overview

This project implements a modular Python-based text processing system that:

- Generates 1000 lines of sample text
- Splits text into chunks
- Performs rule-based sentiment analysis
- Uses parallel processing (ThreadPoolExecutor)
- Stores results in SQLite database
- Provides search and CSV export functionality
- Measures execution time
